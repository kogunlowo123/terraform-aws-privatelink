# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-07

### Added

- Initial release of terraform-aws-privatelink module
- VPC Endpoint Service (producer side) with NLB and GWLB support
- VPC Endpoints (consumer side) for Interface and Gateway types
- Allowed principal management for endpoint service access control
- VPC Endpoint Connection Accepter for manual acceptance workflows
- Default security group creation for Interface endpoints
- VPC Endpoint Policy resource for fine-grained IAM access control
- Private DNS name support for endpoint services
- Basic, advanced, and complete usage examples
- Comprehensive documentation with architecture diagrams
